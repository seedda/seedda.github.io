使い始めるまでに必要そうな知識

TWのメモ

特徴

  • 基本的に一人で使う用のwikiでオフラインの方が多分主流
  • 一般的なwikiとはかなり違うため最初は面食らうかも
  • カスタマイズしやすくする仕組みがある程度用意されている
  • htmlの1ファイルに全てが詰まっており、オフラインならばサーバーは必要ない

難点

  • wikiのような差分/履歴機能が標準で存在しない(連携する機能によってはあるかもしれない)
  • Markdownとは書式が異なる(Markdown用のプラグインはあります)
  • 日本語情報が若干少なめかも？(あっても古いバージョンだったり)

最初にやること

ファイルDL→ブラウザのアドオン等、TWを使いやすくする物を入れる→必要に応じて日本語プラグインを入れる

ファイルDL

公式サイトを開いて少しスクロールして「GettingStarted」と書いてあるところの「Download Empty」をクリックしてファイルをダウンロードする

アドオン等を入れる

そのまま使うと毎度ファイルをダウンロードしていく形になり面倒なのでアドオン等何かしらの補助が必要になる(オンラインはまた別です)
ダウンロードの少し下の「PLATFORMS:」で自分の環境にあった物をクリックして良さそうなのをインストールする方が良い
PCなら以前は「TiddlyFox」(ただし、古いFirefoxのみ対応)、最近なら「Timimi」あたりが良さそう？(試してない)

準備が出来たらhtmlをブラウザで開いて保存ボタンで保存が出来てるか確認してください
ダウンロードになる場合はちゃんと設定できていませんので、アドオンの設定などを確認してください(ブラウザにアドオンのボタンが追加されてないかも)

日本語プラグイン

サイドバーのをクリックで「$:/ControlPanel」を開いて
「Plugins]→青のボタン「Get more plugins」→緑のボタン「open plugin library」→「Languages」→「Japanese(Japan)」→「install」
プラグインをインストールしたら保存してブラウザを更新した後反映される
再度をクリックして「Info」→「Basics」で下の方に「Hello! Current language:」の右に「English～」と書いてあるセレクトボックスがあるのでそこを「Japanese(Japan)」に変更
直後に日本語表示に切り替わる(ほとんどの場所は)

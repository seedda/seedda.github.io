Markdown

TWのメモ

日本語化とほぼ同様の手順で公式のMarkdown parserプラグインを入れる(LanguagesタブでなくPluginsタブにある)

方言が2種類あるが、デフォルトの方言はたぶん表がないので、$:/config/markdown/dialectをGruberからMarukuに変更

以降、WikiTextのTyped Blocksを利用して局所的にMarkdownを利用する

$$$.md
内容
$$$

または

$$$text/x-markdown
内容
$$$

Tiddlerの種類でMarkdownを指定しても使えるが、TWの各種機能が使えなくなる(多分)ため使い勝手はいまいち
